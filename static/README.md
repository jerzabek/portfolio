# Portfolio Image Assets

Place your images in this directory:

- `profile.jpg` - Your profile photo for the sidebar
- `projects/project-1.jpg` - Project 1 screenshot
- `projects/project-2.jpg` - Project 2 screenshot  
- `projects/project-3.jpg` - Project 3 screenshot

You can use placeholder images from services like:
- https://via.placeholder.com/400x300
- https://picsum.photos/400/300
- https://placehold.co/400x300
